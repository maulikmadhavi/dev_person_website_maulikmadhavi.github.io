## Developer Portfolio Landing Page Template

This project provides a simple, customizable landing page template for developers and data scientists, built on the Minimal Jekyll theme for GitHub Pages. It enables users to quickly set up a professional portfolio website using Markdown, with easy editing and previewing directly on GitHub. While the project is discontinued and no longer maintained, you are welcome to fork the repository and adapt it to your needs.

---

### Installation

See the step-by-step tutorial [on Medium](https://medium.com/@evanca/set-up-your-portfolio-website-in-less-than-10-minutes-with-github-pages-d0efa8ff56fd).

You can use the GitHub editor to maintain and preview your website content in Markdown files. Whenever you commit, GitHub Pages will run [Jekyll](https://jekyllrb.com/) to rebuild your site from the Markdown files.
